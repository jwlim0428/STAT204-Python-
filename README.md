## STAT204

## Python Statistics Practice

이 repository는 통계계산프로그래밍 Python의 총 내용중 수업의 기말 범위였던  
CH5–CH7 내용을 바탕으로, Python 데이터 분석의 핵심 개념들을  
개인적으로 다시 정리하며 학습한 기록입니다.

- 데이터 구조 이해 (CH5)
- 조건 처리와 함수화를 통한 사고 정리 (CH6)
- 데이터 시각화를 통한 분석 결과 표현 (CH7)

이라는 흐름으로 정리하며,  
각 코드가 **왜 필요한지**, **어떤 상황에서 사용되는지**를 중심으로 공부했습니다.

아직 대규모 프로젝트 경험은 없지만,  
기초 단계에서 데이터 분석의 전체 흐름을 이해하는 데에 목표를 두고  
실습과 정리를 병행했습니다.
